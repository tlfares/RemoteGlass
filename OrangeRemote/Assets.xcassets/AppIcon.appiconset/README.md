# App Icon

Replace the PNG files in this folder with your final app icon exports.

The most important source is `AppIcon-1024.png`, used for App Store / marketing.
For a production icon, export the same design at these exact pixel sizes:

- `AppIcon-20@2x.png`: 40 x 40
- `AppIcon-20@3x.png`: 60 x 60
- `AppIcon-29@2x.png`: 58 x 58
- `AppIcon-29@3x.png`: 87 x 87
- `AppIcon-40@2x.png`: 80 x 80
- `AppIcon-40@3x.png`: 120 x 120
- `AppIcon-60@2x.png`: 120 x 120
- `AppIcon-60@3x.png`: 180 x 180
- `AppIcon-1024.png`: 1024 x 1024

Keep the filenames unchanged, or update `Contents.json` if you rename them.
